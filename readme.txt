Display your site visitors recent searches easily.

== Description ==
Bring the power of your site search feature and store recent visitors searches. 
Embed the recent searches in the sidebar using a widget or in the content using shortcodes. 
This plugin will displays the most search keywords with a link. If current user is an administrator,
the search keyword can be deleted easily from the front end.

<h4>Key Features & Options</h4>
<ul>
<li>Supports multi instances widget and shortcodes.</li>
<li>Store a search keyword only if greater than 2 characters.</li>
<li>Number of searches to show.</li>
<li>Store only if resulted posts found.</li>
<li>Search keyword validation.</li>
<li>Easy deletion from front end.</li>
</ul>

Read more at <a href="http://www.ground6.com/wordpress-plugins/recent-searches/"><strong>plugin homepage</strong></a>.

== Installation ==
You can use the built in installer and upgrader, or you can install the plugin manually.
<ul>
<li>Go to the menu 'Plugins' -> 'Install' and search for 'Recent Searches'</li>
<li>Click 'install'</li>
</ul>
